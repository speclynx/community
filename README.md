GitHub Discussions has to be enabled on a specific repository, so this repo
serves as that dedicated home — it exists to host **SpecLynx Discussions** and
give the community a single, neutral place to ask questions, share ideas, and
propose features, rather than tying conversations to any one tool's repo.

Alongside Discussions, this repository also holds **community-related
materials** — shared resources that belong to the community as a whole rather
than to a single project.
